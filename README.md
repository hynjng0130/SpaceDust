## SpaceDust
* http://hynjng0130.dothome.co.kr/20190116/ex01.html

### 개요
* HTML 페이지

### 설명
* 태양 이미지를 중심으로 지구 이미지가 원을 그리며 이동하여 공전합니다.
* 지구 이미지에서는 총 세 개의 아이콘이 반복되어 나타나 지구의 자전을 보여줍니다.
* 달 이미지가 지구 이미지 주위를 공전합니다.

### 기술
* HTML
* CSS
* JavaScript
* Visual Studio Code
